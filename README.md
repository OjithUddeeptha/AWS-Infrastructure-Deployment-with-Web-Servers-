# AWS-Infrastructure-Deployment-with-Web-Servers-
AWS Infrastructure Deployment with Web Servers using a custom VPC. Configured subnets, Internet Gateway, route tables, security groups, EC2 (Ubuntu &amp; Amazon Linux) with Apache, S3 bucket, IAM roles, Target Group, and Application Load Balancer. Successfully hosted and validated a load-balanced web application on AWS.
